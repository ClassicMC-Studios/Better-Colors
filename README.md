# Better<a href="">Colors</a>

### A css <a href="">color</a> framework

> <b>Basics</b>

```jsx
  import './bc.css'
  
  function basicReactComponent(){
    return(
      <h1 className='bc-green'></h1>
    );
  };
  
  export default basicReactComponent;

```
